## 简介

德力西 DEP2-025 是符合 GB/T 11918（等效 IEC 60309）的工业级五芯插座，32A/415V，IP44 防水防尘，红色外壳代表三相 380~415V 电压等级，用于实验台三相电源接入。

正视插座面（有孔一侧），引脚按以下位置分布（"6h" 表示钥匙槽在 6 点钟方向）：

```
            ┌─────────────────────────────┐
            │                             │
            │          ⊕  PE              │
            │        (黄/绿 接地)          │
            │                             │
            │   N              L1         │
            │ (蓝 零线)    (棕 A相)        │
            │                             │
            │     L3        L2            │
            │  (灰 C相)  (黑 B相)          │
            │                             │
            │    [▼ 钥匙槽 6h 位置 ▼]      │
            └─────────────────────────────┘
             （面向插座孔面，正视图）
```

**导线颜色对应（IEC/GB 标准）：**

| 引脚 | 颜色 | 说明 |
|------|------|------|
| PE   | 黄/绿双色 | 保护接地，必须首先连接 |

| L1   | 棕色 | A 相（第一相） |
| L2   | 黑色 | B 相（第二相） |
| L3   | 灰色 | C 相（第三相） |

## 接线注意事项

> ⚠️ **危险：三相 415V 高压，接线前务必确认主开关（WHT35）已断开并挂牌锁定。**

1. **接线顺序**：PE（接地）→ N（零线）→ L1/L2/L3（相线），拆线顺序相反
2. **线径要求**：32A 电流对应最小线径 6mm²（铜芯），建议使用带颜色标识的多芯软电缆
3. **力矩要求**：接线端子螺丝拧紧力矩约 2.5~3 N·m，防止接触不良发热
4. **插拔限制**：**禁止带载插拔**，拔出前先断开负载侧断路器
5. **防护维护**：IP44 防护，使用后检查防水密封圈是否完好，避免进水
