## 简介

ITECH IT-7900EP 是一款基于 SiC 技术的全 4 象限再生型可编程电网模拟器，**实物已于 2026-04-29 到货登记**。采用高功率密度设计（21kVA/3U），可在**源模式**（Source）下模拟各类电网异常，也可在**负载模式**（Load）下作为可编程交流/直流电子负载使用。产生的能量可回馈至电网（再生型），适用于并网逆变器、储能系统、微电网、V2X 等设备的研发和合规测试，同时支持 PHiL（功率硬件在环）仿真。

## 源模式（Source Mode）功能详解

### 基本输出模式

支持四种输出组合：

| 模式 | 说明 |
|------|------|
| **AC** | 纯正弦交流输出（默认） |
| **DC** | 纯直流输出 |
| **AC+DC** | 交流叠加直流偏置（DC 偏置在 Config 设置） |
| **DC+AC** | 直流基础上叠加交流纹波（Vac ≤ 额定电压 10%） |

相位配置：

- **单相**：标准单相输出
- **三相**：三相四线（Y 接或 △ 接可选）
- **反相（Reverse Phase）**：两台单相串联，额定电压扩展至 200%（最高 700V L-N）
- **多通道**：3 路独立单相输出，可同时测试 3 台 DUT（高级功能不可用）

---

### 电压跌落与骤升（Surge & Sag）

用于模拟电网暂态异常，可在任意输出波形上叠加：

| 参数 | 说明 |
|------|------|
| **触发模式** | 周期（Period）/ 触发（Trig）两种 |
| **起始角度** | 0–360° 可设，精确控制跌落发生的相位 |
| **角宽度** | 跌落持续的相位角范围 |
| **对称性** | 可生成对称跌落波形 |
| **幅值设定** | 百分比（% of RMS）或绝对值（V）两种方式 |
| **重复次数** | 自定义重复执行次数 |
| **周期计数** | 每 N 个电网周期执行一次（配合 Repeat count） |
| **三相独立控制** | 可选 A / B / C / A&B / B&C / A&C / A&B&C，同步或指定相位模式 |

> **实验室典型用法**：LVRT 测试时，设置 A&B&C 同步跌落至额定的 20%，持续 25 个周期（500ms at 50Hz），对应 IEC 61000-4-11 Class III 要求。

---

### 谐波注入与分析

| 功能 | 参数 |
|------|------|
| **谐波次数** | 最高 50 次（基波 50/60 Hz） |
| **内置标准库** | IEC 61000-3-2（家用设备）、IEC 61000-3-12（工业设备） |
| **THD 波形** | 直接设置输出电压 THD 值，自动合成失真波形 |
| **用户自定义** | 逐次设置各次谐波幅值与相位，支持 CSV 导入 |
| **谐波测量** | 内置仪器实时显示前 50 次谐波幅值与相位 |
| **谐波分析** | 触摸屏频谱图显示 |

---

### 可编程输出阻抗

模拟电网内阻抗（等效改变系统 SCR）：

- **可设参数**：R（阻性）和 L（感性）
- **用途**：无需更换外部接线，直接在仪器内调节 R+L，模拟从强网（低阻抗）到弱网（高阻抗）的不同场景
- **实验室场景**：配合 LCL 滤波器（L1=6mH, L2=300μH），通过增大输出 R+L 模拟 SCR=2~5 的弱网，寻找逆变器失稳临界点

---

### IEC 标准合规测试

内置以下 5 个标准的测试曲线，一键执行：

| 标准 | 测试内容 |
|------|---------|
| **IEC 61000-4-11** | 电压跌落（Voltage Dips）、短时中断（Short Interruptions）、电压变化（Voltage Variations） |
| **IEC 61000-4-13** | 谐波与间谐波（Harmonics & Interharmonics） |
| **IEC 61000-4-14** | 电压波动（Voltage Fluctuation） |
| **IEC 61000-4-17** | 直流纹波（DC Ripple on AC Supply） |
| **IEC 61000-4-28** | 频率变化（Variation of Frequency） |

参数可定制：起始相位角、重复次数、延迟时间、上升/下降斜率。

## 负载模式（Load Mode）功能详解

### AC 负载模式（6 种工作模式）

| 模式 | 说明 |
|------|------|
| **CC**（恒流） | 吸收恒定有效值电流 |
| **CP**（恒功率） | 吸收恒定功率 |
| **CR**（恒阻） | 模拟纯阻性负载 |
| **CS**（恒视在功率） | 恒定视在功率 |
| **CC+CR** | 恒流与恒阻复合模式 |
| **CE**（电路仿真） | 模拟真实电路拓扑（见下） |

**CE 模式支持的电路拓扑**：
- **并联 RLC**：R、L、C 并联后接地，模拟谐振型负载
- **单相整流 RLC**：整流桥 + RLC 滤波，模拟开关电源前级整流器

其他 AC 负载特性：
- 波峰因数（Crest Factor）：1.414–5.0 可调
- 相移：-180°~+180° 可设
- 单位功率因数模式（Unit PF）：电流随电压变化，PF 趋近 1

### DC 负载模式

支持 9 种工作模式：CC / CR / CP / CV / CC+CV / CR+CV / CP+CV / CC+CR / CV+CC+CP+CR 自动模式。

## 反孤岛测试模式（Anti-islanding Test）

专用于验证并网逆变器的反孤岛保护功能：

1. 进入 **RLC 功能菜单**，配置等效负载 R、L、C 参数
2. 或直接设定孤岛平衡点的**有功/无功功率**值
3. 设置孤岛发生时的输出停止相位角
4. **开启 K1 开关**，切换至孤岛状态
5. 界面**实时显示孤岛持续时间**，测量逆变器保护响应时间

适用标准：IEEE 1547、IEC 62116 反孤岛检测验证。

## LIST / SWEEP / 任意波形

### LIST 函数

- 最多 **200 步**序列，每步独立设置波形、幅值、频率、运行时间、上升斜率
- 11 种内置波形 + 用户自定义波形均可作为子波形
- 支持 **USB 导入 CSV 文件**快速编辑
- 可设置循环次数和结束状态（保持最后波形 / 关闭输出 / 返回正常）

### SWEEP 函数

- 电压 + 频率双维度扫描，逐步阶梯式变化
- 扫描结束自动标记**最大功率点**（适合 MPPT 效率测试）
- 支持时间触发和外部触发两种切换方式
- 扫描模式：单向 / 来回扫描（back-forth）

### 内置及自定义波形

| 类型 | 波形 |
|------|------|
| 内置 8 种 | 正弦、方波、锯齿波、三角波、梯形波、削顶正弦、整流波、THD 波 |
| THD 波 | 直接设定 THD 比例，自动合成失真正弦 |
| 用户自定义 | 手动编辑 or CSV 导入，保存至本地波形库 |

自定义波形可作为 LIST / Surge-Sag / SWEEP 的子波形使用。

## PHiL 功率放大器模式

IT-7900EP 可配置为 PHiL（Power Hardware-in-the-Loop）功率放大器：

- **工作原理**：外部实时仿真器（如 OPAL-RT OP4200）输出低压模拟/数字信号，IT-7900EP 将其无失真放大为真实功率电压/电流
- **接口要求**：需选配**模拟接口（Ext-Program / Analogue Function）**；标配 Digital I/O 可作触发同步
- **实验室拓扑**：OPAL-RT → 模拟信号 → IT-7900EP 功率放大 → LCL 滤波器 → Danfoss FC302 逆变器（DUT）
- **应用场景**：微电网实时仿真、储能 BMS 测试、新能源车 V2X（BOBC）功率硬件验证

## 快速使用

### 场景 1：LVRT 低电压穿越测试

1. 设置输出为**三相 AC**，220V L-N，50Hz，使能输出
2. 进入 **Standard → IEC 61000-4-11**，选择 Voltage Dips 测试项
3. 设置跌落深度（如 20% of Un），持续周期数（如 25 cycles = 500ms）
4. 确认起始角度（通常 0°），点击 **Run** 执行
5. 外接示波器或采集系统记录 DUT 响应电流波形

### 场景 2：谐波注入测试

1. 设置单相/三相 AC 正常输出
2. 进入 **Config → Waveform → THD Wave**，输入目标 THD 百分比（如 5%）
3. 或在谐波设置界面逐次设置 3、5、7 次谐波幅值与相位
4. 使能输出，内置**谐波分析仪**实时验证输出波形 THD
5. 测量 DUT 在谐波污染电网下的响应（电流畸变、保护动作等）

### 场景 3：可编程阻抗模拟弱电网

1. 进入 **Config → Impedance**，设置 R 和 L（如 R=0.5Ω，L=5mH，模拟 SCR≈5）
2. 设置正常 AC 输出（220V L-N，50Hz），使能输出
3. 观察逆变器在弱网阻抗条件下的稳定性（PLL 锁定时间、电流谐波等）
4. 逐步增大 R+L（降低 SCR），寻找失稳临界点

## 注意事项

- **大功率接线**：三相运行时电流可达 30A（L-N），接线前确认配电开关容量 ≥ 40A，线缆截面积 ≥ 6mm²，输出端必须可靠接地
- **能量回馈**：负载模式吸收的能量将回馈至输入电网；确认上游配电允许双向潮流，防止触发过压保护
- **SiC 散热**：保持设备前后通风不受遮挡（≥ 50cm）；环境温度不超过 40°C；预热 30 分钟后测量精度最佳
- **反相模式**：700V L-N 高压输出时，必须使用额定电压 ≥ 1000V 的测试线，并确认 DUT 耐压等级
- **多通道限制**：三路独立输出时，Surge&Sag、LIST、IEC 合规测试等高级功能均不可用
- **功率型号核对**：本页记录为 IT7921EP（21kVA）；若实验室实际采购为其他功率等级，请以机身铭牌为准
