## 简介

LCL 滤波器是并网逆变器与电网之间的无源接口，用于衰减逆变器开关谐波，使并网电流满足 IEEE 1547 / GB/T 14549 谐波标准。本滤波器为实验室 FYP 项目定制设计的三相 PCB，配合 Danfoss VLT FC302 逆变器使用。**实物已于 2026-04-29 到货登记。**

## 电路拓扑图

```
  逆变器侧 (Danfoss 输出)                电网侧 (IT-7900EP)

  A ──[L1a: 6mH]──┬──[L2a: 300μH]── A'
                  │
               [Rd: 1Ω]
               [Cf: 10μF]
                  │
  B ──[L1b: 6mH]──┼──[L2b: 300μH]── B'
                  │
               [Rd: 1Ω]
               [Cf: 10μF]
                  │
  C ──[L1c: 6mH]──┴──[L2c: 300μH]── C'
                  │
               [Rd: 1Ω]
               [Cf: 10μF]
                  │
                 GND
```

> 每相电容支路：Rd（1 Ω）与 Cf（10 μF）**串联**后接地，Rd 为无源阻尼电阻，抑制 LCL 谐振峰。

**各元件作用：**

| 元件 | 参数 | 作用 |
|------|------|------|
| L1（逆变器侧电感） | 6 mH | 限制开关纹波电流，保护逆变器 |
| Cf（滤波电容） | 10 μF | 提供谐波电流旁路，与 L2 形成谐振陷波 |
| Rd（阻尼电阻） | 1 Ω（与 Cf 串联） | 无源阻尼，抑制 LCL 谐振峰，防止谐振不稳定 |
| L2（电网侧电感） | 300 μH | 阻挡高频谐波进入电网，与 Cf 形成截止频率 |

## 谐振频率

LCL 谐振频率（无阻尼时）：

```
f_res = (1/2π) × √[(L1 + L2) / (L1 × L2 × Cf)]
      = (1/2π) × √[(6mH + 300μH) / (6mH × 300μH × 10μF)]
      = (1/2π) × √[6.3×10⁻³ / 1.8×10⁻¹¹]
      ≈ (1/2π) × 18708 rad/s
      ≈ 约 3.0 kHz
```

阻尼电阻 Rd = 1 Ω 引入的阻尼比（以单相等效估算）：

```
ζ ≈ Rd / 2 × √[Cf / (L1 ‖ L2)] ≈ 0.1 ~ 0.3（轻阻尼，谐振峰被抑制约 10–15 dB）
```

> ⚠️ 控制带宽应低于谐振频率的 1/3（约 1.0 kHz），否则控制器会激励谐振导致不稳定。Rd 提供无源阻尼，但会引入额外功耗（约 Rd × Icap² per phase）。

## 元件对应实验室库存

| LCL 元件 | 参数 | 对应实验室耗材词条 |
|----------|------|-----------------|
| L1 | 6 mH | 铁芯环形电感 2mH × 3 串联，或专绕 6mH 环形电感（`inductor-toroid-2mh`） |
| L2 | 300 μH | 铁芯环形电感 ~80μH × 4 串联，或专绕 300μH（`inductor-toroid-80uh`） |
| Cf | 10 μF | 需采购 10μF 薄膜电容（现有 `cap-film-4u7-400v` 可 2 只并联 ≈ 9.4μF 应急） |
| Rd | 1 Ω | 功率电阻（Icap 峰值约 2–3A，需 ≥ 5W 金属膜电阻）— 当前库存未录入 |

## 注意事项

- 上电前检查所有电感绕组绝缘，环形电感匝间绝缘靠漆包线，注意引脚焊点处不要破损
- 滤波电容额定 400V，三相 380V 系统峰值约 537V，**必须使用 400V 以上薄膜电容**，不可使用铝电解电容（交流纹波过大）
- PCB 设计文件（Altium Designer `.SchDoc` / `.PcbDoc`）存放于坚果云实验室文件服务器
